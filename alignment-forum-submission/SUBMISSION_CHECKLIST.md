# Alignment Forum Submission Checklist

Use this checklist to ensure your post is ready for submission.

## Pre-Submission Setup

- [ ] Alignment Forum account created and verified
- [ ] Markdown editor enabled in account settings
- [ ] All 9 images located and ready to upload
- [ ] GitHub repository link verified and accessible
- [ ] All references checked for accuracy

## Content Preparation

- [ ] Read through `subliminal-preference-transfer.md` completely
- [ ] Verify all LaTeX equations are correct
- [ ] Check all links (GitHub, references) work
- [ ] Review abstract for clarity and completeness
- [ ] Ensure discussion questions are clear and engaging

## Image Handling

- [ ] Located all 9 images from original blog post
- [ ] Images are in acceptable format (PNG/JPG)
- [ ] Images are appropriately sized (< 5MB each)
- [ ] Prepared descriptive alt text for each image
- [ ] Know where images will be uploaded

## Formatting Check

- [ ] All headings use proper Markdown syntax (`#`, `##`, `###`)
- [ ] Math equations use `$$` for block math
- [ ] Inline math uses single `$` (if needed)
- [ ] Lists are properly formatted
- [ ] Tables are correctly formatted
- [ ] Code blocks (if any) are properly formatted
- [ ] No frontmatter (YAML header) remains

## Content Review

- [ ] Abstract accurately summarizes the research
- [ ] Introduction clearly states the research question
- [ ] Methodology is clearly explained
- [ ] Results are accurately presented
- [ ] Discussion questions are thought-provoking
- [ ] References are complete and properly formatted
- [ ] Conclusion ties everything together

## Technical Verification

- [ ] LaTeX renders correctly (test in preview)
- [ ] All markdown syntax is valid
- [ ] No broken links
- [ ] Tables display correctly
- [ ] Code blocks (if any) are readable

## Alignment Forum Specific

- [ ] Title is clear and descriptive
- [ ] Tags selected: `AI Safety`, `RLHF`, `Preference Learning`, `Interpretability`, `Fairness`, `Alignment`
- [ ] Post is set to draft initially (for final review)
- [ ] Preview looks good on desktop
- [ ] Preview looks good on mobile (if possible)

## Final Steps

- [ ] Copy entire content from `subliminal-preference-transfer.md`
- [ ] Paste into Alignment Forum editor (Markdown mode)
- [ ] Upload all 9 images and replace placeholders
- [ ] Add title and tags
- [ ] Preview the entire post
- [ ] Check all images display correctly
- [ ] Verify LaTeX math renders properly
- [ ] Read through one final time
- [ ] Save as draft
- [ ] Get feedback from co-author (if desired)
- [ ] Publish when ready!

## Post-Publication

- [ ] Share on social media (if desired)
- [ ] Monitor comments and respond thoughtfully
- [ ] Update GitHub repository with link to post
- [ ] Consider cross-posting to LessWrong (if applicable)

## Troubleshooting

If you encounter issues:

**LaTeX not rendering:**
- Verify Markdown editor is enabled
- Check that equations use `$$` for block math
- Try escaping special characters if needed

**Images not displaying:**
- Verify image URLs are correct
- Check that images are publicly accessible
- Try re-uploading if necessary

**Formatting issues:**
- Use preview mode frequently
- Check Alignment Forum's editor guide
- Verify markdown syntax is correct

**Other issues:**
- Check Alignment Forum's help documentation
- Reach out to Alignment Forum moderators if needed
