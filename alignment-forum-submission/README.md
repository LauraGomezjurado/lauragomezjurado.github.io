# Alignment Forum Submission Guide

This folder contains an Alignment Forum-ready version of the blog post "Subliminal Preference Transfer in LLMs: When Models Learn More Than We Intend".

## Files

- `subliminal-preference-transfer.md` - The main blog post, formatted for Alignment Forum
- `SUBMISSION_CHECKLIST.md` - Step-by-step checklist for posting
- `IMAGE_GUIDE.md` - Instructions for handling images

## Quick Start

1. **Enable Markdown Editor** on Alignment Forum:
   - Go to your account settings
   - Under "Site Customizations", check "Activate Markdown Editor"

2. **Prepare Images**:
   - See `IMAGE_GUIDE.md` for details
   - You'll need to upload images separately and update the image paths in the markdown

3. **Copy Content**:
   - Open `subliminal-preference-transfer.md`
   - Copy the entire content
   - Paste into Alignment Forum's new post editor (make sure Markdown mode is enabled)

4. **Add Metadata**:
   - Title: "Subliminal Preference Transfer in LLMs: When Models Learn More Than We Intend"
   - Tags: `AI Safety`, `RLHF`, `Preference Learning`, `Interpretability`, `Fairness`, `Alignment`
   - Draft status: Review before publishing

5. **Review and Submit**:
   - Use the preview feature to check formatting
   - Verify all LaTeX math renders correctly
   - Ensure all images are properly linked
   - Check that all links work
