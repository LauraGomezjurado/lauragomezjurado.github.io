# About Page Photos

Place your photos in this directory with the following filenames:

- `microsoft.jpg` - Photo at Microsoft (with Microsoft ID badge)
- `davos.jpg` - Photo at World Economic Forum Davos 2025
- `un.jpg` - Photo at United Nations
- `research.jpg` - Photo presenting research at Stanford

The images will be displayed in a responsive grid gallery on the About page. Recommended aspect ratio is 4:5 (portrait orientation) for best display.
